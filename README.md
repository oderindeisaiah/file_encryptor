# File Encryptor / Decryptor

**Secure File Encryption Tool in Python**  
A Python project demonstrating secure file encryption and decryption using symmetric cryptography (Fernet). Perfect for showcasing cybersecurity skills and portfolio projects.

---

## **Features**

- ✅ Encrypt any file locally with a secure key  
- ✅ Decrypt files safely using the same key  
- ✅ Automatic key generation (`filekey.key`)  
- ✅ Works entirely offline — no servers or network required  
- ✅ Clean, professional code structure for portfolio
